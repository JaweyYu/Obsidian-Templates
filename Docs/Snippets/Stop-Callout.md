## 使用方法

首先请阅读：[CSS Snippets 的安装方法](../Usages/Install-CSS-Snippets.md)。

然后按如下格式使用：

```markdown
> [!hr-stop] 工事中，立入禁止
```

标题文字随意修改，不过不要太长。

也可以加入更多内容，效果也不错：
